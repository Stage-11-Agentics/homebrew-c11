# Homebrew Tap for c11

This is the official Homebrew tap for [c11](https://github.com/Stage-11-Agentics/c11), a native macOS terminal multiplexer for AI coding agents.

## Installation

```bash
brew tap stage-11-agentics/c11
brew install --cask c11
```

## Updating

```bash
brew upgrade --cask c11
```

## Uninstalling

```bash
brew uninstall --cask c11
brew untap stage-11-agentics/c11
```
